# wp llms students-progress update

Updates an existing students-progress.

### OPTIONS

&lt;id&gt;
: The id for the resource.

[\--id=&lt;id&gt;]
: Unique identifier for the student. The WP User ID.

[\--post_id=&lt;post_id&gt;]
: Unique course, lesson, or section Identifer. The WordPress Post ID.

[\--date_created=&lt;date_created&gt;]
: Creation date. Format: Y-m-d H:i:s

\--status=&lt;status&gt;
: The status of the enrollment.
\---
options:
  - complete
  - incomplete
\---

[\--porcelain]
: Output just the id when the operation is successful.
